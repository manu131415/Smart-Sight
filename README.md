# 👁️ SmartSight – AI-Based Assistive Navigation System for the Visually Impaired

## 📌 Overview
SmartSight is an intelligent assistive system designed to help visually impaired individuals navigate safely and independently. The project combines embedded systems and smart sensing technologies to detect obstacles, identify water hazards, and provide real-time alerts along with location tracking.

This system enhances traditional mobility aids by integrating automation, communication, and future AI capabilities.

## 🚀 Features
- Obstacle Detection using Ultrasonic Sensor
- Water Hazard Detection using Water Sensor
- Real-Time Location Tracking using GPS (NEO-6M)
- Emergency Alert System via GSM module (SMS)
- Audio Alerts through buzzer for immediate feedback
- Low-cost and portable design

## 🧠 Future AI Enhancements
- Voice-based feedback system
- Smart path prediction using Machine Learning
- Mobile app integration for live tracking
- Fall detection using accelerometer
- Object recognition using camera (AI vision)

## 🛠️ Tech Stack
- Hardware: Arduino, Ultrasonic Sensor, Water Sensor, GPS NEO-6M, GSM Module, Buzzer
- Software: Arduino IDE (Embedded C/C++)
- Communication: GSM (SMS alerts), GPS (location tracking)

## ⚙️ Working Principle
- Ultrasonic sensor continuously scans for obstacles
- If an object is detected within a threshold distance → buzzer alerts user
- Water sensor detects moisture → triggers warning alert
- GPS module tracks real-time location
- GSM module sends emergency SMS with location to predefined contacts
